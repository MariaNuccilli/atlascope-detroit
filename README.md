# Atlascope Starter

> A starter kit for instantiating Atlascope in your town, from @bplmaps

## What's Atlascope?

Atlascope is a discovery tool for exploring georeferenced urban atlas like Sanborn maps. Developed and maintained by the [Leventhal Map & Education Center](https://leventhalmap.org), 

## Basic concepts

Setting up Atlascope for your town mostly happens in the `instance.json` file.

## Building locally
